# Leetcode面接高頻出問題まとめ

参考: https://zhuanlan.zhihu.com/p/349940945

200問以上、派生問題含めて約250問。ほとんどがmedium、少数のhard。アルゴリズム面接には十分。

---

**ソート:**
- 基礎: Quick Sort & Merge Sortの実装、計算量、In-place、安定性
- 入門: Leetcode 148, 56, 27
- 応用: Leetcode 179, 75

**選択:**
- Quick Selection、Heapsort。トレードオフを理解し、C++ STLの対応メソッドを知る。
  - Leetcode 215, 347, 2099
  - Leetcode 4. Median of Two Sorted Arrays

**連結リスト:**
- 基礎: 実装と走査。先頭/末尾の挿入削除はO(1)、検索はO(N)
- 入門: Leetcode 206, 876

快慢ポインタとリスト反転は基本。反転コードは暗記推奨。

- 応用: Leetcode 160, 141, 92, 328

**Heap, Dictionary, Stack, Queue:**
- Queue: Leetcode 225, 346, 281, 1429, 54, 362
- Stack: Leetcode 155, 232, 150, 224, 20, 1472, 1209, 1249, 735
