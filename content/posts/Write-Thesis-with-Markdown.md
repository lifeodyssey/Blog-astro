# Initial Setup

这里参考的是[这个](https://github.com/lowercasename/docdown/wiki/Initial-setup)。

在装好zotero和Better BibTex之后，在文件那里选导出，格式选择**BetterBibTeX JSON**，除了Keep updated别的别选。我把他放在onedrive里面了。

然后去属性里打开Better BibTex,把Automatic export改成on change。

# Cite while write

这里需要[这个](https://github.com/davepwsmith/zotpick-applescript)，一个简单的方法是直接用vscode，我就直接用vscode然后把Citation Picker for Zotero下载下来就完事了。

然后按一下shift +option+z就可以呼唤出zotero

# Pandoc

还是直接在vscode里装好pandoc， 直接cmd+K+P完事

更详细的可以看[这里](https://chuhong.life/007-markdown-to-docx-pdf/#VScode-%E6%8F%92%E5%85%A5%E5%BC%95%E6%96%87)

反正看完了这一圈之后，我感觉，就写个毕业论文，字数不多，暂时就没空改了。

等后面再写别的慢慢培养习惯。
