为了解决在国内使用claude的问题，进行了一番研究

<!-- more -->

我给出的解决方案是[5ber](https://esim.5ber.com/)+[sim2fly](https://esim2fly.com/)

# prerequisite

1. 你需要一台能够使用esim的安卓手机，这个可以在这里看到[Supported Devices - 5ber eSIM](https://esim.5ber.com/supportdevice?language=zh-CN)
2. 你需要一个科学上网方式

至于境外支付的方式？不需要，支付宝和微信就行

# 我为什么选择这个方式

我要解决的第一个问题是，我本人并没有在国外的合法居留方式，因此日本和美国被我排除了。

（其实我一开始最想找的是日本的，因为方便接受日本的一堆fan club抽选之类的，但是日本的限制太多了，目前查下来如果在日本没有合法居留方式，最快捷的方式是去日本旅游的时候在当地买esim进行激活；第二种方式是买已经注册好的日本当地手机卡，但是好贵...）。

其次，由于港澳在一些时候也被划分为“不能提供服务”的区域，于是也把港澳的卡给排除了。

在这个前提下，筛选了一下能够进行空中发卡的厂商，我的要求是

1. 便宜
2. 能够境外接受短信
3. 有效期长
   最后只剩下两家,一个是Giffgaff，一个是[ESIM2Fly Shop Home | ESIM2Fly.com](https://esim2fly.com/)。

第二个问题，使用esim还是实体sim卡。

我的回答是使用esim卡，因为实体sim卡需要从海外发货，比较麻烦。

第三个问题，国内基本没有手机支持esim，所以我购买了[5ber](https://esim.5ber.com/)

# 这个方式的缺点

1. 5ber卡必须把卡插进去才能接受短信，并不是完全的esim。
2. esim需要单独购买和付费，5ber并不自带。
3. 一张手机卡只能下载两张esim。
4. 要保号的话，需要固定去充值保号。
5. 由于流量少且昂贵，不适合去旅游的用户。

# 具体流程

1. 在这个[页面](https://esim.5ber.com/order?language=zh-CN)购买，可以直接填国内的收货地址，利用支付宝和微信支付付款，我选择的是普通版本。可以用我的邀请码`W3L9A9`来获得一部分优惠。
   ![image.png](https://raw.githubusercontent.com/lifeodyssey/Figurebed/master/20240720232501.png)

2. 大概5天左右就可以收到sim卡，接下来像使用国内的sim卡一样把它插到手机卡里，下载这个app [5ber.eSIM APK for Android Download](https://apkpure.com/5ber-esim/com.ifreegroup.moesim)

3. 在打开这个app登陆之后，他会检测你的手机能否使用5ber卡，一切顺利的话是这个界面。 
   
   ![image.png](https://raw.githubusercontent.com/lifeodyssey/Figurebed/master/20240720232912.png)

4. 这个时候就需要你去找一张esim的卡来扫描了，来到[sim2fly](https://esim2fly.com/)这个网站，我买的是红框里的套餐，正常购买支付宝或者微信付款就好
   
   ![image.png](https://raw.githubusercontent.com/lifeodyssey/Figurebed/master/20240720233128.png)

5. 接下来你的邮箱会收到一个二维码以及你的手机号，那个esim code就是你的手机号。
   ![image.png](https://raw.githubusercontent.com/lifeodyssey/Figurebed/master/20240721002235.png)

拿你的5ber app去扫码即可

![image.png](https://raw.githubusercontent.com/lifeodyssey/Figurebed/master/20240720233610.png)

接下来就可以拿它去注册claude了，不过第一次接受短信延迟会比较久。

![image.png](https://raw.githubusercontent.com/lifeodyssey/Figurebed/master/20240720233654.png)

# 问题

- Q：如何保号？
- A：可以下载[my ais](https://apkpure.com/myais/com.ais.mimo.eservice)进行充值。 6个月之后充值一次可以保号一个月。

- Q：如何查找其他合适的esim卡？
- A：可以直接谷歌搜索对应的国家+esim或者咸鱼碰碰运气，注意要使用你想要的国家的语言。也可以在下面几个网站搜一下：
  - [Buy travel eSIMs for your next trip | MobiMatter](https://mobimatter.com/)  
  - [Jetpac | Travel eSIM](https://www.jetpacglobal.com/us/) 
  - [Compare eSIM for Asia - Find the best prepaid data plans for travelers](https://esimdb.com/region/asia) 

- Q: 除了claude  ，我还可以用来干嘛

- A：可以用来注册line和whats app，我也尝试了yahoo japan id，但是目前看来yahoo japan id还是只支持日本境内的手机号。

- Q: claude真的比gpt4好用吗

- A：我问了同一个问题，可以参考一下，个人感觉和gpt4基本是同一水平

问题

***I want to remove file 20240720233343.png in the github both local and remotely permanently, no body can see it in the git history, can you give me a one line git command to do it***

gpt 4

```bash
git filter-branch --force --index-filter "git rm --cached --ignore-unmatch 20240720233343.png" --prune-empty --tag-name-filter cat -- --all && git push origin --force --all && git push origin --force --tags
```

gpt 4o

```bash
git filter-repo --path 20240720233343.png --invert-paths --force
```

gpt 4o mini

```bash
git filter-repo --path 20240720233343.png --invert-paths
```

claude 3.5 sonnet

```bash
git filter-branch --force --index-filter 'git rm --cached --ignore-unmatch 20240720233343.png' --prune-empty --tag-name-filter cat -- --all && git push origin --force --all
```
