# 備考

CSSスタイルが効かない時のトラブルシューティング：

1. ブラウザでF12を押してCSSソースがローカルと一致するか確認
2. 一致する場合、キャッシュをクリアして再確認
3. CSSが一致しキャッシュの問題でもない場合、詳細度かタイプミスの可能性が高い
